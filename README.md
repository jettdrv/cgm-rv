# Runtime Verification di un APS - Artifical Pancreas System

## Struttura:
- 'spec.pdf': contiene tutte le informazioni utili sulle scelte implementative e le specifiche in STL
- 'data/': file csv generati da Simglucose
- 'results/': risultati del monitor
- 'controller/': controller personalizzato basato sulla predizione
- 'monitor/': continene gli script per la generazione degli eventi a partire da un file csv e il monitor
- 'grafici/': grafici della robustezza delle specifiche di accuratezza creati durante l'esecuzione del monitor
